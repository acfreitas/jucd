JUCD
====

## Introdução

Jovens Unidos Contra as Drogas

## Requisitos

* [VirtualBox](https://www.virtualbox.org) 
* [Vagrant](http://vagrantup.com)

## Setup

    $ git clone https://github.com/JUCD/jucd.git
    $ cd jucd
    $ vagrant up

## Teste

	$ [http://localhost:3000](http://localhost:3000)
